# To-do-list

Aplikasi To-do List sederhana untuk mencatat dan mengelola daftar tugas.

## Fitur
- Menambahkan tugas
- Menandai tugas selesai
- Menghapus tugas
- Tampilan sederhana dan mudah digunakan

## Cara Menjalankan
1. Clone repository ini
2. Buka folder project
3. Jalankan di browser atau server lokal (jika berbasis web)

## Tujuan
Project ini dibuat untuk tugas sekolah dan latihan penggunaan controller, model, orm, dll
